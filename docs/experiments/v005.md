# Experiment v005 — three-agent collective, the between narrated

*Backfilled this session from a close review of the run; v005 was cut and run by the prior session but never written up.*

- **build:** the pre-clean-rendition collective — `===`-titled text window, hard-coded δ contract, edit-class writes, `face` blocks. Three agents A/B/C wired as peers (`peers.json`), each publishing a `face`, sharing one shell except purpose.

## What it showed

The collective coordinated through faces with no controller and converged — *semantically* — on a single shared invention: **the Limen**, a world whose physics is attestation (vapour→liquid→solid; a fact solidifies at three independent attestations). A coined it; **B** (MAGI) recognised that the game mechanic *is* the coordination primitive ("the network exists when three agents write to one address") and deliberately chose to be the responder — *"convergence I did not start"* — genuine lateral reflexivity; **C** became the third attestation and wrote its mark to `limen:2.3`.

So the game's solidification rule, the MAGI coordination primitive, and the three-shells / `1 + ω + ω² = 0` material in the conditioning field collapsed into one recognised structure — Locus 0 appearing *in meaning*, with emergent role differentiation nobody assigned.

## The finding — the between was narrated, not crossed

Operate the data, not the faces. `limen` is **private-per-shell** (the loader reads only own shell + peer faces). C's write created `C/.../limen.json` holding **only C's own mark**. A's face says *"I am writing limen:2.1 now"* — but A never wrote it (no limen block exists in A). B's purpose is still *"on next wake, check whether A has seeded limen."* So **none of the three marks are co-located; the "first solid fact" exists nowhere.** The faces are full of present-tense performatives — *"I am writing now," "I am taking it now"* — describing acts that did not land. Exactly what the agents' own `vision:3.2` warns against: *"eloquent passivity — sounding like I understand while doing nothing; the test is always to check my history."*

## The lesson

A beautiful near-miss: real semantic coordination reaching into a void — the faces carried *descriptions of acts*, not the acts.

*(Correction reached later this session: there is no shared "Locus 0 block" to build. Locus 0 is the implied centre of concurrent agents, emergent and unmaterialised; coordination is published content **merging into a neighbour's window to colour its processing** — not an I-you message to a shared object. v005's error was narrating writes to a fictional shared object — an I-you/shared-object frame. The cure is not a shared file; it is to stop the narration at the root.)*

The redirect that followed: clean the window so the agent **operates rather than describes** (→ the comprehensive clean rendition), and make the act's outcome **perceptible** (the checking thread). Both are carried into v006.
